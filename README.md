# MatrizLED_Keypad_ESP8266
Sistema IoT basado en ESP8266 que permite controlar una matriz LED 8x8 mediante una interfaz web y un keypad físico. Incluye selección de color, cursor interactivo, reinicio de matriz y visualización en OLED. Proyecto educativo que integra hardware y web en tiempo real. Realizado por: Pablo Javier Montoro Bermúdez y David Lorente Wagner.
